# KARNAGE

A demoscene-inspired real-time AV demo and a test bench for the `infinite*` ecosystem (infinitedsp, infinitegfx, and infinitemedia). 

Everything is generated on the fly and synchronized to demonstrate how the crates work together.

## Run it
```bash
cargo run --release
```

Or for web (using Trunk):
```bash
cd karnage
trunk serve --release
```
